# Turtle Flowers

Cleaned up some code I dug up from 2017 when I was first learning Python and generated some images (instead of being stuck inside the tkinter window)

Turtle Flowers based off of [Think Python 2 chapter 4](https://greenteapress.com/thinkpython2/thinkpython2.pdf).

# Getting Started

Make sure you have pipenv installed with your local Python installation
```
pip install pipenv
```

Install packages
```
pipenv install
```

Run the main file
```
pipenv run start
```
